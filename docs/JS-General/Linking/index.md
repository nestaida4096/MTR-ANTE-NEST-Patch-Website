# 連結機能について

!!! warning "実験的機能"
    この機能は、まだ開発中です。利用の際は、不具合などに特に注意してください。

    なお、解結機能はバグが残っており非推奨です。

## 連結
### 主な用途
- 列車同士を途中駅で増結する
### 連結モード
- `0`: 通常時に使用するモード。新たに連結を行う状態ではない。
- `1`: 連結モード（**連結する側**）
- `2`: 連結モード（**連結される側**）
- `-1`: 解結モード
### 連結方法
1. 車両を2つ用意し、1つは駅の定位置に停車させる。
2. もう１つの車両は、駅の手前で停止させ、`/train toggleCollision`を実行する（TransitManagerが必要）。
3. 低速で車両に接近させ、2つの車両同士が衝突した後1m程度進んで停止させる。
4. 手動運転の場合、前の車両に移動しドアの開閉を行う（連結成功の場合後ろの車両もドアの開閉が行われる）。
5. 連結完了
