# MTR-ANTE NEST Patch 利用ルール
## 目的
- このルールは、リソースの悪意ある利用や不適切な利用を防ぎ、安心して利用するために制定します。
- 利用者は、このルールを必ず遵守する必要があります。
## 禁止事項（やってはいけないこと）
- **ソースコードをデコンパイル/リバースエンジニアリングなど、解析した結果を他のユーザーに共有する行為**
- 事故を表現する行為やシステムのバグに起因した事故を撮影する行為（記録に残す行為）
- Modの二次配布などの不正な公開行為（サーバー内配布については制作者が定める手続きが必要です）
    - **mrpackなどのmodパックの内部にjarデータが内包されている場合も不正な公開として扱います**
## 免責事項（やってもいいけど何かあっても制作者は責任をとりません）
- ソースコードをデコンパイル/リバースエンジニアリングなど、解析する行為
- 通信メソッド（Request系など）の使用レートが高いまま処理を実行する行為
